**Постановка задачи**
Прогноз стоимости медицинской страховки человека на основе его персональных данных.

Датасет: https://www.kaggle.com/datasets/mohankrishnathalla/medical-insurance-cost-prediction/data

Датасет представлен csv-файлом из **54 столбцов и 100K строк**

**Названия столбцов с пояснением:**

*   **'person_id'**: Уникальный идентификатор для каждого человека
*   **'age'**: Возраст человека
*   **'sex'**: Пол
*   **'region'**: Географический регион
*   **'urban_rural'**: Городской пейзаж
*   **'income'**: Годовой доход (в долларах США)
*   **'education'**: Уровень образования
*   **'marital_status'**: Семейное положение
*   **'employment_status'**: Тип/статус занятости
*   **'household_size'**: Количество человек в домохозяйстве
*   **'dependents'**: Количество финансовых иждивенцев
*   **'bmi'**: Значение индекса массы тела
*   **'smoker'**: Курение (да/нет)
*   **'alcohol_freq'**: Частота употребления алкоголя
*   **'visits_last_year'**: Количество обращений к врачу за последний год
*   **'hospitalizations_last_3yrs'**: Количество госпитализаций за последние три года
*   **'days_hospitalized_last_3yrs'**: Общее количество дней, проведенных в больнице за последние три года
*   **'medication_count'**: Количество назначенных активных лекарственных препаратов
*   **'systolic_bp'**: Систолическое артериальное давление (мм рт. ст.)
*   **'diastolic_bp'**: Диастолическое артериальное давление (мм рт. ст.)
*   **'ldl'**: Уровень холестерина ЛПНП (мг/дл)
*   **'hba1c'**: Процентное содержание гликированного гемоглобина (показатель контроля уровня сахара в крови)
*   **'plan_type'**: Тип страхового плана (например, HMO, PPO, EPO)
*   **'network_tier'**: Уровень страховой сети (базовый, серебряный, золотой, платиновый) (Basic, Silver, Gold, Platinum)
*   **'deductible'**: Ежегодная сумма франшизы в долларах США
*   **'copay'**: Размер доплаты за медицинскую услугу
*   **'policy_term_years'**: Срок действия страхового полиса в годах
*   **'policy_changes_last_2yrs'**: Количество изменений в политике за последние два года
*   **'provider_quality'**: Рейтинг качества работы первичного медицинского учреждения/страховой компании
*   **'risk_score'**: Рассчитанный показатель риска для здоровья (нормализованный)
*   **'annual_medical_cost'**: Общая годовая стоимость медицинских услуг в долларах США - **ТАРГЕТ**
*   **'annual_premium'**: Ежегодная страховая премия в долларах
*   **'monthly_premium'**: Ежемесячная страховая премия в долларах США
*   **'claims_count'**: Количество страховых случаев, зарегистрированных за год
*   **'avg_claim_amount'**:Средняя сумма иска в долларах США
*   **'total_claims_paid'**: Общая сумма выплаченных компенсаций в долларах США
*   **'chronic_count'**: Количество диагностированных хронических заболеваний
*   **'hypertension'**: Показатель диагностированной гипертонии (0 = нет, 1 = да)
*   **'diabetes'**: Показатель наличия диабета (0 = нет, 1 = да)
*   **'asthma'**: Показатель наличия астмы (0 = нет, 1 = да)
*   **'copd'**: Индикатор хронической обструктивной болезни легких (0 = нет, 1 = да)
*   **'cardiovascular_disease'**: Наличие сердечно-сосудистых заболеваний (0 = нет, 1 = да)
*   **'cancer_history'**: Наличие в анамнезе онкологического заболевания (0 = нет, 1 = да)
*   **'kidney_disease'**: Индикатор наличия хронической болезни почек (0 = нет, 1 = да)
*   **'liver_disease'**:Показатель заболевания печени (0 = No, 1 = Yes)
*   **'arthritis'**: Показатель наличия артрита (0 = нет, 1 = да)
*   **'mental_health'**: Наличие диагностированного психического расстройства (0 = нет, 1 = да)
*   **'proc_imaging_count'**: Количество процедур визуализации (например, рентгенографии, МРТ, КТ), проведенных за год
*   **'proc_surgery_count'**: Количество перенесенных хирургических вмешательств
*   **'proc_physio_count'**: Количество психиатрических/психологических процедур или сеансов
*   **'proc_consult_count'**: Количество консультаций специалистов (например, кардиолога, онколога)
*   **'proc_lab_count'**: Количество пройденных лабораторных диагностических исследований
*   **'is_high_risk'**: Бинарный индикатор, указывающий на высокий риск для здоровья человека (0 = нет, 1 = да)
*   **'had_major_procedure'**: Указывает, перенес ли человек серьезное заболевание (например, операцию, госпитализацию) за последний год (0 — нет, 1 — да)
